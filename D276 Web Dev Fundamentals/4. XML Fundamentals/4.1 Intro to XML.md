XML (Extensible Markup Language) is a markup language that you can use to structure and format any kind of data that have multiple purposes such as the following:
* data storage
* data retrieval 
* data mining 
* data processing 
way to attach information to text 
XML is a way to provide structure to data but it's not necessarily meant to be rendered
HTML can use XML and in a way HTML is a specific set of XML 
### XML Tag Formatting
Can create your own tags in XML not limited to predefined tags like HTML 
XML Tags follow 1 of 2 formats 
	Element tags:
		`<city>"LA"</city>`
	Self-closing tags:
		`<hierarchy title="United States"/>`
